## 한 줄 요약
Windows 11의 WSA에서 생성되는 아티팩트 체계를 분석하고 수집 도구를 개발하여 Android 앱 사용 행위 복원.

## 핵심 발견
- **userdata.vhdx** 경로: `%UserProfile%\AppData\Local\Packages\MicrosoftCorporationII.WindowsSubsystemForAndroid_8wekyb3d8bbwe\LocalCache` — Android 데이터 전체 저장
- Windows 측 아티팩트: LNK(`%AppData%\Roaming\Microsoft\Windows\Start Menu\Programs`), Prefetch(WSACLIENT.EXE 등), $UsnJrnl(userdata.vhdx 변경 추적), UserAssist, Recent Files, SRUM(`%SystemRoot%\System32\sru\SRUDB.dat`), AmCache, Timeline
- WSA 설정 디렉토리: LocalState(`settings.dat`, Icons, `appcompatdb.json`)
- 수집 방법: ADB(127.0.0.1:58526), SMB 서버, UsageStats
- 커스텀 CLI 수집 도구 개발·공개
