## 한 줄 요약
Windows 아티팩트(Prefetch, Registry, Chrome 기록, MFT)를 자동으로 수집·분석하고 NLP로 웹 기록 키워드화하여 소셜 네트워크 관계 시각화.

## 핵심 발견
- 수집 대상: HOST 시스템 정보, Prefetch, Registry 키, Chrome Login Data/History, MFT 플래그, Chrome Sync
- Mecab NLP 엔진으로 웹 접속 기록 키워드 추출 및 분류
- 소셜 네트워크 분석으로 사용자 행위 패턴 시각화
- 다수 아티팩트 자동 상관 분석으로 수사 효율화 가능성 제시
