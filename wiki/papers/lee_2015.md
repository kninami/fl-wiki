## 한 줄 요약
Windows 8 Style UI 앱(Viber, Line)의 SQLite DB 아티팩트를 분석하고 R/igraph로 통화·메시지 관계망 시각화.

## 핵심 발견
- Windows 8 앱 데이터 경로: `AppData\Local\Packages`
- **Viber DB** (`...\Viber-FreePhoneCalls\...\viber.db`): Events(timestamp, direction, type, Contact lat/lon/number), Messages(status, body, payload_path), ChatInfo, UploadFile, Contact, Origin Number 테이블
- **Line DB** (`...\NAVER.LINEwin8\...\CacheWp`): Chatl(MID, Mid_Type, Display_Name, Status, Last_Modified), Loginl(User_ID) 테이블
- R igraph 패키지로 발신/수신 관계 그래프 생성 → 핵심 연락처 시각적 식별
