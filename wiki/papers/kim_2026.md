## 한 줄 요약
현장 선별 압수 절차를 반영한 3단계(검색·선택·압수) 평가 모델 수립 및 6종 Windows 포렌식 도구의 선별 압수 기능 비교 실험.

## 핵심 발견
- 평가 단계 3개: ①검색(Search) - NTFS 파싱·아티팩트 분석, ②선택(Selection) - 키워드 검색·필터링·암호화 파일 처리, ③압수(Seizure) - 논리 이미지 생성·무결성 검증·보고서
- **검색 단계**: NTFS $MFT 파싱 및 타임스탬프 로그 분석에서 도구별 편차 큼
- **선택 단계**: 파일 필터링·암호화 파일 처리·미리보기 기능 도구별 격차 존재
- **압수 단계**: 증거 수집 무결성·무결성 검증·보고서 요건 충족하는 도구가 소수
- 단일 도구로 모든 요건을 충족하는 도구 없음 → 상황별 도구 조합 필요
- ISO/IEC 9126, NIST CFTT 기준 기반 평가 체계
