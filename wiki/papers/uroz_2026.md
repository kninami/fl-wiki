## 한 줄 요약
Windows Vista~11 전 버전을 지원하는 NT Heap 구조 순회 알고리즘 및 Volatility 3 플러그인(HeapList) 구현.

## 핵심 발견
- NT Heap은 Backend Layer(세그먼트 관리)와 Frontend Layer(LFH, 소형 할당 최적화)로 구성
- Backend: 각 Heap Entry의 Size·Flags 필드가 XOR 인코딩(EncodeFlagMask 기반)으로 난독화 → 디코딩 알고리즘 제시
- Frontend(LFH): Windows 8.1부터 stride와 offset이 RtlpLFHKey로 XOR 인코딩 → 추가 디코딩 필요
- WinDbg 교차 검증 및 Windows Heap API 제어 실험으로 정확성 확인
- Telegram Desktop 케이스스터디: 힙 분석으로 포렌식 관련 데이터(연락처, 메시지 등) 추출 성공
